# 🌤️ SkyGlass Weather App

A sleek, responsive weather application featuring a **Glassmorphism (frosty glass)** design aesthetic, enhanced with pure CSS 3D water drops. Built entirely with vanilla HTML, CSS, and JavaScript, it is lightweight, fast, and optimized for instant deployment via **GitHub Pages**.

## ✨ Features

* **Comprehensive Metrics:** Displays City name, Country, Precise Local Time, Coordinates (Latitude/Longitude), Altitude, Temperature, Humidity, Wind Speed, Precipitation, Air Quality Index (AQI), and Sunrise/Sunset times.
* **Glassmorphic UI:** A beautiful modern layout utilizing `backdrop-filter` blur effects, translucent borders, and dynamic shadows.
* **Pure CSS Water Drops:** Interactive-looking background water droplets styled with complex CSS inner shadows to mimic realism.
* **No API Keys Required:** Uses the free, open-source **Open-Meteo API** (both Geocoding and Weather endpoints), making it perfectly secure to host publicly without leaking environment variables.
* **Timezone Intelligent:** Automatically computes and updates the exact current local time for the searched destination, regardless of where you are searching from.
